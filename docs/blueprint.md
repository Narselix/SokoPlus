# **App Name**: SokoPlus

## Core Features:

- User Account Management: Secure user account creation, login, and profile management with role-based access control (User, Seller, Trainer, Recruiter, Pharmacist, Admin).
- Education Platform: Online courses with video, text, and PDF content, progress tracking, quizzes, and certifications for local skills development.
- E-commerce Marketplace: Platform for product listings, categories, shopping cart functionality, and email-based order notifications between buyers and sellers.
- Job & Freelance Board: Job postings and freelance opportunities with online applications, professional profiles, and skill-based matching.
- Online Pharmacy: Catalog of medications, partner pharmacies, order placement via notifications, and product information (no medical diagnosis provided).
- Solidarity & Donation Platform: Platform to run campaigns for donations, support war victims and local community projects, offering action transparency.
- AI-Powered Recommendation Tool: Suggest relevant courses, products, job listings, or solidarity campaigns to users based on their profiles and activities.

## Style Guidelines:

- Primary color: Warm orange (#FF7043) to represent the energy and opportunity within the Soko+ ecosystem.
- Background color: Light beige (#F5F5DC), to create a welcoming and neutral backdrop that emphasizes the content.
- Accent color: Teal (#4DB6AC), to signal important interactive elements or notifications, setting them apart from the primary elements.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern, approachable aesthetic, suitable for both headlines and body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, clear icons representing each platform module.
- Clean, modular layout with a central dashboard for unified navigation.
- Subtle animations for module transitions and data loading to enhance user experience.
