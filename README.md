
# Soko+ - L'écosystème numérique du Kivu

Soko+ est une plateforme intégrée révolutionnaire conçue pour dynamiser la communauté du Kivu en regroupant l'éducation (ERP Scolaire), le commerce local, l'emploi et la solidarité.

## 🚀 Fonctionnalités Clés

### 🎓 Module Éducation (ERP Scolaire de Type Shule)
Un système complet de gestion pour les établissements d'enseignement, adapté à la réalité de la RDC :
- **Gestion Administrative** : Inscription des élèves avec génération de matricules uniques (format: `SOKO-SC-2025-XXXX`).
- **Comptabilité Scolaire** : Configuration des frais (minerval, uniforme, etc.), suivi des tranches de paiement.
- **Facturation & Reçus** : Génération et impression automatique de reçus officiels intégrant le **logo et le cachet officiel** de l'école.
- **Rapports Financiers** : Tableaux de bord pour suivre les recettes par classe, par jour et par mois.
- **Portail Parent/Élève** : Suivi des paiements et téléchargement des reçus.
- **Pédagogie** : Saisie des notes, appel numérique et gestion de la bibliothèque.

### 🛒 Services Communautaires
- **Marché Local** : Plateforme de vente pour les artisans et commerçants.
- **Logement** : Annonces vérifiées de location et de vente immobilière.
- **Emploi & Freelance** : Publication d'offres et de missions pour les talents locaux.
- **Transport** : Mise en relation pour la mobilité urbaine.

### 🏥 Santé & Bien-être
- **Annuaire Santé** : Localisation des hôpitaux et cliniques.
- **Espace Écoute** : Soutien psychologique via des articles et un forum anonyme.

### 🤖 Intelligence Artificielle (Genkit)
- **Conseils Personnalisés** : Suggestions de cours, de produits et d'opportunités basées sur le profil utilisateur.

## 🛠️ Stack Technique
- **Frontend** : Next.js 15 (App Router), React 19, Shadcn UI, Tailwind CSS.
- **Backend** : Firebase (Authentication, Firestore, Storage).
- **IA** : Firebase Genkit (Google Gemini 2.5 Flash).
- **Gestion d'État** : React Hook Form + Zod.

## 📦 Installation

1. **Cloner le projet** :
   ```bash
   git clone <votre-url-github>
   cd soko-plus
   ```

2. **Installer les dépendances** :
   ```bash
   npm install
   ```

3. **Lancer le mode développement** :
   ```bash
   npm run dev
   ```
